Examples: generating jsPanels

Example No 1.1
The easiest way to create a jsPanel without any configuration at all.
$('#content').jsPanel();
Example No 1.2
Add a configuration object to set position and size of the jsPanel. 
$('#content').jsPanel({
    position: { top: 50, left: 475 },
    size: { width: 500, height: 310 }
});
Example No 1.3
Add id and title to the configuration object/jsPanel. 
$('#content').jsPanel({
    id: 'example13',
    title: 'Example No 1.3',
    position: { top: 150, left: 475 }
});
Example No 1.4
Remove the scrollbars from the jsPanel. 
$('#content').jsPanel({
    id: 'example14',
    title: 'Example No 1.4',
    position: { top: 300, left: 475 },
    overflow: { vertical: 'hidden', horizontal: 'hidden' }
});
Example No 1.5
Position the jsPanel centered within the parent element of the jsPanel. 
$('#content').jsPanel({
    id: 'example15',
    title: 'Example No 1.5',
    size: { width: 500, height: 310 },
    position: { top: 'center', left: 'center' }
});

Examples: adding content

Example No 2.1
Adding content using the content option with a HTML string. 
$('#content').jsPanel({
    id: 'example21',
    title: 'Example No 2.1',
    position: { top: 50, left: 75 },
    size: { width: 'auto', height: 'auto' },
    content: '<p>A simple HTML string as content</p>'
});
Example No 2.2
Adding content using the content option with a function. 
function bsp22(){
    return "<p>Return value of a function</p>";
}
$('#content').jsPanel({
    id: 'example22',
    title: 'Example No 2.2',
    position: { top: 150, left: 75 },
    size: { width: 'auto', height: 'auto' },
    content: bsp22()
});
Example No 2.3
Adding content using the content option with an Immediately Invoked Function Expression. 
$('#content').jsPanel({
    id: 'example23',
    title: 'Example No 2.3',
    position: { top: 300, left: 50 },
    size: { width: 'auto', height: 'auto' },
    content: (function(){
        return "<p>Content returned from an IIFE.</p>";
    }())
});
Example No 2.4
Adding content using a callback function. 
$('#content').jsPanel({
    id: 'example24',
    title: 'Example No 2.4',
    position: { top: 570, left: 360 },
    size: { width: 700, height: 'auto' }
},
function(){
    $('.jsPanel-content', '#example24' ).load( 'files/example24.html' );
});
Example No 2.5
Adding content using option.ajax. 
$( '#content' ).jsPanel({
    id: 'example25',
    title: 'Example No 2.5',
    position: { top: 'center', left: 780 },
    ajax: {
        url: 'files/example25.html'
    }
});
Example No 2.6
Adding content using option.load. 
$( '#content' ).jsPanel({
    id: 'example26',
    title: 'Example No 2.6',
    position: { top: 'center', left: 780 },
    load: {
        url: 'files/example26.html',
        complete: function(){ console.log( "It's done!" ) }
    }
});

Examples: adding extras & styling

Example No 3.1
Adding an extra toolbar. 
$('#content').jsPanel({
    id: 'example31',
    title: 'Example No 3.1',
    position: { top: 50, left: 345 },
    toolbarContent: '<img src="jsPanel/images/cut.png"> ...'
});
Example No 3.2
Adding a preloader image. 
$('#content').jsPanel({
    id: 'example32',
    title: 'Example No 3.2',
    position: { top: 190, left: 345 },
    content: "<div class='preloader'><img src='js/jsPanel/images/pl.gif'></div>"
});
Example No 3.3a
Set css background of the content area using the contentBG option with a string. Uses jQuery.css( 'background' , option.contentBG ) 
$('#content').jsPanel({
    id: 'example33a',
    title: 'Example No 3.3a',
    position: { top: 315, left: 345 },
    contentBG: '#65A5D1'
});
Example No 3.3b
Set css background of the content area using the contentBG option with an object. Uses jQuery.css( option.contentBG ). Offers an option to set more than just background properties. 
$('#content').jsPanel({
    id: 'example33b',
    title: 'Example No 3.3b',
    position: { top: 480, left: 345 },
    contentBG: { 'background': '#A62D00' }
});
Example No 3.4
Setting options for the jQuery-UI resizable & draggable features. 
$('#content').jsPanel({
    id: 'example34',
    title: 'Example No 3.4',
    position: { top: 590, left: 345 },
    resizable: {
        minWidth:   300,
        minHeight:  150,
        maxWidth:   800,
        maxHeight:  600
    },
    draggable:  {
        stop: function(event, ui) {
            ui.helper.css( 'background', '#235849' );
        }
    }
});

Childpanel within a parentpanel. 
$('#content').jsPanel({
    id: 'example42',
    title: 'Example No 4.2',
    position: { top: 'center', left: 'center' }
    },function(){
        $( '#example42 .jsPanel-content' ).jsPanel({
            size: {width: 150,height: 100}
        });
});
